# chest-menu-gen-v2

Generiert eine .png von dem oberen Menü einer Kiste, gefüllt mit den ausgewählten items.

Requirements:
- Python 3.11
- Pillow (Python Library, pip install Pillow)
- CustomTkinter (Python Library, pip install customtkinter)
